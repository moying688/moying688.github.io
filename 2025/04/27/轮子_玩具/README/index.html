目前我们就是，启动五个 Raft服务，然后各自启动自己的选举计时器

当超时的时候，Term变为1，然后发起选举，失败后，重置计时器，此时其他的结点超时Term变为1，但是此时他们只能捡漏第一个结点没有拿到的票，

如果都失败了，再次等待超时进入第二轮直到有新的Leader
